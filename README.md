# LFP_Processor
A Graphical User Interface built on Python and QT framework that can process Local Field Potential data.

The ideal file format to use with this is a MATLAB .mat file with two variables:  
LFP - LFP Data  
t - time data

Matplotlib  
Numpy  
Scipy  
PyQt5
